# CIS051-2-Assignment-Week-1
Assignment Week-1

This Assignement week 1 shows basic use of html5 element tags for building simple webpage.


## Project Files
1. **basic.html**  
   Demonstrates basic HTML elements including headings (`h1`, `h2`, `h3`), ordered lists, unordered lists, and `span` elements.

2. **form.html**  
   Contains an HTML form with input fields, radio buttons, checkboxes, a textarea, and a submit button.

3. **table.html**  
   Displays an HTML table styled with CSS, including borders, a caption, and merged cells for better organization.

4. **motorbikes.html**  
   A complete webpage about motorbikes using HTML5 semantic elements such as `header`, `nav`, `article`, `aside`, and `footer`.

## Steps to Run the Project
1. Download or copy all four HTML files into a single folder.
2. Open any file (`.html`) using a modern web browser.
3. No additional software, libraries, or server setup is required.
